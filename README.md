## Docker
[Download Docker Desktop](https://www.docker.com/get-started)

Build containers by typing in console:
```console
docker-compose build
```

To run my app type:
```console
docker-compose up
```

### Running locally (alternative to docker)

Install node modules:
```console
npm install
```

And run app by:
```console
npm run dev
```
